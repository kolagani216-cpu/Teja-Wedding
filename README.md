# Wedding Invitation App

A beautiful, interactive wedding invitation React application built with Vite, optimized for mobile devices.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

The app will be available at http://localhost:3000

## Features

- Interactive door-opening animation (double-click to open)
- Live countdown timer
- Hero section with couple's names
- Event details and location links
- Photo gallery with placeholders
- RSVP form with feedback
- Mobile-first responsive design
- Built with React 18 and Vite
